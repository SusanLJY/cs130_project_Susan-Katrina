User Guide

Here is an explanation of the functionailities of this website.

Welcome Page

  • Functionalities
    • Show user guide and project design ideas
  • Link to other pages
    • Cover Page (button “Start your journey now”)
    • Quick links to diary, calendar, and monthly schedule

Cover Page

  • Functionalities
    • Let the user choose the target year and month for the schedule.
      • Change the year by clicking either button “<”/”>”, or the smaller years.
  • Link to other pages
    • Welcome Page (button “My daily planner”)
    • Monthly Schedule (button “Target month” and the year in the middle.)

Monthly Schedule

  • Functionalities
    • Add events on the calendar.
      • Empty event detection.
    • Switch between different months and years.
    • Save and load the page in the browser's web storage API.
      • Auto-loading when entering the page.
      • No auto-saving when leaving the page.
    • Let the user go to the target dairy.
  • Link to other pages
    • Cover Page (button “Back to Main Menu”)
    • Dairy (select target date and input a string containing the text “dairy”, click “save”)

Diary

  • Functionalities
    • Making a dairy
      • Scrollable text box.
      • Photo gallery with thumbnail.
      • Image upload (max. 3)
          • Only available for downloading the package, opening with live server, and putting target images in the folder “./Dairy/images”.
      • Save and load target day's dairy. (No auto-saving/loading)
  • Link to other pages
    • Cover Page (button “Main Menu”)
    • Monthly Schedule (button “Date”)
